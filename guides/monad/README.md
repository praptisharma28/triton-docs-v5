# Guides by Monad Labs

Building on Monad? These official guides from Monad Labs take you from your first deploy through indexing. Monad is EVM-compatible, so your existing Ethereum tooling works.

1. Deploy a contract from the browser, no local setup: [**deploy with Remix**](https://docs.monad.xyz/guides/deploy-smart-contract/remix)
2. Deploy using the Foundry toolchain: [**deploy with Foundry**](https://docs.monad.xyz/guides/deploy-smart-contract/foundry)
3. Deploy using Hardhat: [**deploy with Hardhat**](https://docs.monad.xyz/guides/deploy-smart-contract/hardhat)
4. Publish your source so explorers can read it: [**verify a contract with Foundry**](https://docs.monad.xyz/guides/verify-smart-contract/foundry)
5. The same, via Hardhat: [**verify with Hardhat**](https://docs.monad.xyz/guides/verify-smart-contract/hardhat)
6. Wire wallet connection into your dApp: [**connect a wallet with Reown AppKit**](https://docs.monad.xyz/guides/reown-guide)
7. A shareable on-chain action link: [**build a donation Blink**](https://docs.monad.xyz/guides/blinks-guide)
8. Index on-chain events: [**index transfers with GhostGraph**](https://docs.monad.xyz/guides/indexers/ghost)
9. Stream indexed data into an app: [**index for a Telegram bot with Envio**](https://docs.monad.xyz/guides/indexers/tg-bot-using-envio)
10. A managed streaming indexer: [**index with QuickNode Streams**](https://docs.monad.xyz/guides/indexers/quicknode-streams)
