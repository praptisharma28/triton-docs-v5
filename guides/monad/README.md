# Guides by Monad Labs

Building on Monad? These official guides from Monad Labs take you from your first deploy through indexing. Monad is EVM-compatible, so your existing Ethereum tooling works.

1. [**Deploy with Remix**](https://docs.monad.xyz/guides/deploy-smart-contract/remix) — deploy a contract from the browser, no local setup.
2. [**Deploy with Foundry**](https://docs.monad.xyz/guides/deploy-smart-contract/foundry) — deploy using the Foundry toolchain.
3. [**Deploy with Hardhat**](https://docs.monad.xyz/guides/deploy-smart-contract/hardhat) — deploy using Hardhat.
4. [**Verify a contract with Foundry**](https://docs.monad.xyz/guides/verify-smart-contract/foundry) — publish your source so explorers can read it.
5. [**Verify with Hardhat**](https://docs.monad.xyz/guides/verify-smart-contract/hardhat) — the same, via Hardhat.
6. [**Connect a wallet with Reown AppKit**](https://docs.monad.xyz/guides/reown-guide) — wire wallet connection into your dApp.
7. [**Build a donation Blink**](https://docs.monad.xyz/guides/blinks-guide) — a shareable on-chain action link.
8. [**Index transfers with GhostGraph**](https://docs.monad.xyz/guides/indexers/ghost) — index on-chain events.
9. [**Index for a Telegram bot with Envio**](https://docs.monad.xyz/guides/indexers/tg-bot-using-envio) — stream indexed data into an app.
10. [**Index with QuickNode Streams**](https://docs.monad.xyz/guides/indexers/quicknode-streams) — a managed streaming indexer.
