# Guides by Mysten Labs

Building on Sui? These official guides from Mysten Labs take you from first setup to full applications.

1. Get the toolchain on your machine: [**install the Sui CLI**](https://docs.sui.io/getting-started/onboarding/install-binaries)
2. Connect to a network and set up your keys: [**configure the Sui client**](https://docs.sui.io/getting-started/onboarding/configure-sui-client)
3. Fund a devnet or testnet address: [**get test SUI from the faucet**](https://docs.sui.io/getting-started/onboarding/get-coins)
4. Write, build, and publish your first Move package: [**Hello, World**](https://docs.sui.io/getting-started/onboarding/hello-world)
5. The map of what to learn next: [**developer guides overview**](https://docs.sui.io/guides/developer/getting-started)
6. How Sui represents state, the core concept to internalise: [**the Sui object model**](https://docs.sui.io/guides/developer/objects)
7. Programmable transaction blocks (PTBs): [**building transactions**](https://docs.sui.io/guides/developer/transactions)
8. RPC, gRPC, GraphQL, and indexers: [**accessing on-chain data**](https://docs.sui.io/guides/developer/accessing-data)
9. Signing, multisig, and zkLogin: [**cryptography**](https://docs.sui.io/guides/developer/cryptography)
10. Full Move and React builds to learn from: [**end-to-end example apps**](https://docs.sui.io/getting-started/examples)
