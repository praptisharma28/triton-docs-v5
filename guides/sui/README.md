# Guides by Mysten Labs

Building on Sui? These official guides from Mysten Labs take you from first setup to full applications.

1. [**Install the Sui CLI**](https://docs.sui.io/getting-started/onboarding/install-binaries) — get the toolchain on your machine.
2. [**Configure the Sui client**](https://docs.sui.io/getting-started/onboarding/configure-sui-client) — connect to a network and set up your keys.
3. [**Get test SUI from the faucet**](https://docs.sui.io/getting-started/onboarding/get-coins) — fund a devnet or testnet address.
4. [**Hello, World**](https://docs.sui.io/getting-started/onboarding/hello-world) — write, build, and publish your first Move package.
5. [**Developer guides overview**](https://docs.sui.io/guides/developer/getting-started) — the map of what to learn next.
6. [**The Sui object model**](https://docs.sui.io/guides/developer/objects) — how Sui represents state, the core concept to internalise.
7. [**Building transactions**](https://docs.sui.io/guides/developer/transactions) — programmable transaction blocks (PTBs).
8. [**Accessing on-chain data**](https://docs.sui.io/guides/developer/accessing-data) — RPC, gRPC, GraphQL, and indexers.
9. [**Cryptography**](https://docs.sui.io/guides/developer/cryptography) — signing, multisig, and zkLogin.
10. [**End-to-end example apps**](https://docs.sui.io/getting-started/examples) — full Move and React builds to learn from.
