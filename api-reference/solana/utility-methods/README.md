# Utility methods

Read node and cluster information that is not tied to a specific account or transaction. These methods return epoch and slot details, cluster nodes, node identity and version, inflation, supply, the leader schedule, snapshot slots, and health.

Use this section for diagnostics, monitoring, and reading network-level parameters. `requestAirdrop` is available on test clusters only.
